TYPETR Presti is a sophisticated serif typeface inspired by the flexible-pen contrast of traditional expansion stroke models, echoing the refined style of Bodoni with a modern twist. With its wide range of weights and optical sizes, Presti adapts beautifully across both print and digital media, from delicate captions to bold headlines. Rich in OpenType features, it offers a stunning array of stylistic alternates, flourishes, and typographic finesse, making it ideal for editorial design, branding, and luxury packaging.

Multiple swashes on one character

The [ss01] feature needs InDesign 19.0, later than October 2023

The feature [ss01] is using a code sequence to connect swashes to characters.

  • Start with the character itself
  • Directly followed by a “tilde” ~
  • Followed by the code for vertical position: “t”: top, “m”: middle, “b”: bottom
  • Followed by the code for the horizontal position: “l”: left, “r”: right
  • Followed by the code for the vertical direction: “u”: up, “d”: down
  • finally a single digit that defines the size of the swash: “1”: small, “2”: larger, “3”: moderate, “4”: large, “5”: super sized
  • A~tru3 – A~tru3
  • B~mrd2 – B~mrd2
  • C~bld5 – C~bld5
  • D~mru5 – D~mru5
  • E~mlu1 – E~mlu1

Several code sequences can be combined to attach multiple swashes to the same character. Each character has up to 8 connections points.

  • A~tru3~bru5 – A~tru3~brd3
  • B~mrd2~tlu2 – B~mrd2~tlu2
  • C~bld5~tru3 – C~bld5~tru3
  • D~mru5~mrd3 – D~mru5~mrd3
  • E~mlu1~tlu2~brd2 – E~mlu1~tlu2~brd2
